shadcn/ui is a collection of re-usable components built using Radix UI and Tailwind CSS. It's not a component library — it's a collection of components that you can copy and paste into your apps.

Follow these steps to get started with shadcn/ui in your project:

  1. Run npx shadcn@latest init
  2. Choose your style and base color
  3. Add components with npx shadcn@latest add [component]
  4. Import and use in your app

shadcn/ui uses CSS variables for theming. You can customize colors, radius, and more. Edit your globals.css to override:

:root {
  --background: oklch(1 0 0);
  --primary: oklch(0.205 0 0);
  --radius: 0.625rem;
}
The accordion follows the WAI-ARIA Accordion Pattern. Each trigger has aria-expanded, aria-controls, and the content panel has role="region" with aria-labelledby. Keyboard navigation with Space and Enter is supported.

Can I use this in production?

Yes, absolutely. shadcn/ui is production-ready and used by thousands of projects.

Does it support TypeScript?

Yes. All components are written in TypeScript with full type safety.

Can I use it without Tailwind?

No. Tailwind CSS v4+ is required. However you own the code so you can adapt it.
